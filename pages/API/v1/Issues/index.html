Issues are a way for Content Owners and Translation Resources to communicate about translation work. Each Issue is attached to one unique string in a Smartling project. An issue is either Open or Closed and can have any number of user comments attached. For more information about how Issues work in the Smartling Dashboard, see our user documentation.
